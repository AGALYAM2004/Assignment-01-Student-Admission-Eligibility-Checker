# Assignment-01-Student-Admission-Eligibility-Checker

## Project Description

This Python program is a simple **Student Admission Eligibility Checker** for a training institute.

The program collects student details and checks whether the student is eligible for admission based on their percentage.

---

## Features

- Accepts student details:
  - Student Name
  - Age
  - Degree
  - Percentage

- Displays student information

- Checks admission eligibility:
  - Percentage >= 60 → Eligible
  - Percentage < 60 → Not Eligible

- Calculates additional information:
  - Years remaining to reach age 30
  - Percentage required to reach 100

---

## Concepts Used

- Variables
- Data Types
- User Input
- Output Statements
- Type Casting
  - int()
  - float()

- Operators
  - Comparison operator (>=)
  - Arithmetic operator (-)

- Conditional Statements
  - if-else

---

## Program Workflow

1. Get student details from the user
2. Display student information
3. Check percentage eligibility
4. Display admission status
5. Calculate additional details
